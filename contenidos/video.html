Video

El < video > elemento tiene una serie de atributos que le permiten controlar la reproducción de vídeo:

Atributos de video

src

Este atributo especifica la ruta de acceso al vídeo. (El video de ejemplo está en formato H264, por lo que solo funcionará en IE y Safari).

poster

Este atributo le permite especificar una imagen para que se muestre mientras se descarga el vídeo o hasta que el usuario le indique al vídeo que se reproduzca.

width, height

Estos atributos especifican el tamaño del reproductor en píxeles.

controls

Cuando se utiliza, este atributo indica que el explorador debe proporcionar sus propios controles para la reproducción.

autoplay

Cuando se utiliza, este atributo especifica que el archivo debe reproducirse automáticamente.

loop

Cuando se utiliza, este atributo indica que el vídeo debe comenzar a reproducirse de nuevo una vez que haya finalizado.

prelaod

Este atributo le dice al navegador qué hacer cuando se carga la página. Puede tener uno de estos tres valores:

none

El navegador no debe cargar el vídeo hasta que el usuario pulse reproducir.

auto

El navegador debería descargar el video cuando se cargue la página.

metadata

El navegador solo debe recopilar información como el tamaño, el primer fotograma, la lista de pistas y la duración.

type

Debes usar este atributo para decirle al navegador qué formato tiene el video. De lo contrario, descargará parte del video para ver si puede reproducir el archivo (lo que llevará tiempo y ancho de banda).

dentro de type también va el Codec, se refleja con un punto y coma
