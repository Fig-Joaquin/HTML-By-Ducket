Estructura básica de la Tabla

< th >

< th > Sirve para crear la tabla. El contenido se escribe fila por fila

< tr >

Indica el comienzo de cada fila de la tabla

< td >

Cada celda de la tabla es representada por el uso del < td >

< th >

El < th> elemento se usa igual que el < td> elemento, pero su propósito es representar el encabezado de una columna o una fila.

Uso de tablas largas

< thead>

Los encabezados de la tabla deben estar dentro del < thead> elemento

< tbody >

The body should sit inside the < tbody> element.

< tfoot >

The footer belongs inside the < tfoot > element
