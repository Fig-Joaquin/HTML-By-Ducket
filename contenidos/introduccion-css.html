Introducción al CSS

En esta sección, veremos cómo hacer que tus páginas web sean más atractivas, controlando el diseño de las mismas mediante CSS.

CSS le permite crear reglas que especifican cómo debe aparecer el contenido de un elemento. Por ejemplo, puede especificar que el fondo de la página sea crema, que todos los párrafos aparezcan en gris con el tipo de letra Arial o que todos los encabezados de nivel uno estén en azul, cursiva, tipo de letra Times.

Entendiendo el css

La clave para entender cómo funciona CSS es imaginar que hay un cuadro invisible alrededor de cada elemento HTML.

Bloques y elementos en linea

Los elementos de nivel de bloque parecen comenzar en una nueva línea. Algunos ejemplos son los
< h1>< h6> , < p> y < div> .
Los elementos en línea fluyen dentro del texto y no comienzan en una nueva línea. Algunos ejemplos son:
< b>, < i>, < img>< em> y < span>.

Ejemplos de estilos

Boxes

Anchura y altura Bordes (color, anchura y estilo) Color de fondo e imágenes Posición en la ventana del navegador.

Texto

Tamaño de la tipografía Color Cursiva, negrita, mayúsculas, minúsculas, versalitas

Específico

También hay formas específicas en las que puede aplicar estilo a ciertos elementos, como listas, tablas y formularios.


CSS asocia reglas de estilo con elementos HTML

CSS funciona asociando reglas con elementos HTML. Estas reglas rigen cómo se debe mostrar el contenido de los elementos especificados. Una regla CSS contiene dos partes: un selector y una declaración.

Partiendo con CSS

Hay tres formas de aplicar CSS a un documento HTML: CSS externo, CSS interno.

CSS Interno

Se usa dentro de la misma página del HTML

CSS Externo

Se usa en un archivo externo al CSS

Elemento

< link >

El elemento se puede usar en un documento HTML para decirle al navegador dónde encontrar el archivo CSS utilizado para dar estilo a la página. Es un elemento vacío (lo que significa que no necesita una etiqueta de cierre) y vive dentro del < head> elemento. Debe usar tres atributos:

href

Esto especifica la ruta al archivo CSS (que a menudo se coloca en una carpeta llamada css o estilos).

type

Este atributo especifica el tipo de documento al que se va a vincular. El valor debe ser text/css

rel

Esto especifica la relación entre la página HTML y el archivo al que está vinculada. El valor debe ser la hoja de estilo cuando se vincula a un archivo CSS. Una página HTML puede usar más de una hoja de estilo CSS. Para ello, podría tener un < link> elemento para cada archivo CSS que utilice. Por ejemplo, algunos autores utilizan un archivo CSS para controlar la presentación (como fuentes y colores) y un segundo para controlar el diseño
