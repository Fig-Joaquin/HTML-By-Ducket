Audio

< audio >

El < audio > elemento tiene una serie de atributos que le permiten controlar la reproducción de audio:

src

Este atributo especifica la ruta de acceso al archivo de audio.

controls

Este atributo indica si el reproductor debe mostrar los controles. Si no utiliza este atributo, no se mostrará ningún control de forma predeterminada. También puede especificar sus propios controles mediante JavaScript.

autoplay

La presencia de este atributo indica que el audio debe comenzar a reproducirse automáticamente. (Se considera una mejor práctica permitir que los visitantes elijan reproducir audio).

preload

Este atributo indica lo que debe hacer el navegador si el reproductor no está configurado para la reproducción automática. Puede tener los mismos valores que vimos en la página 214 para el < video> elemento.

loop

Este atributo especifica que la pista de audio debe reproducirse de nuevo una vez que haya finalizado.

Multiples fuentes de una canción

< source >

Es posible especificar más de un archivo de audio utilizando el < source> elemento entre las etiquetas de apertura < audio> y cierre (en lugar del atributo src en la etiqueta de apertura < audio> ). Esto es importante porque los diferentes navegadores admiten diferentes formatos para los archivos de audio.

type

Se define el tipo de contenido, en este caso es audio y el formato es mp3.
