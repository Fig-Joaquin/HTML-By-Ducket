Estas propiedades incluyen:

  • Crear bordes alrededor de las cajas

  • Mostrar y ocultar cajas

Dimensiones de la caja

Las dimensiones de las cajas se pueden controlar con las propiedades de width y height en CSS.

Contenido de la caja

Las formas más populares de especificar el tamaño de un cuadro son usar píxeles, porcentajes o ems. Tradicionalmente, los píxeles han sido el método más popular porque permiten a los diseñadores controlar con precisión su tamaño.

Cuando se utilizan porcentajes, el tamaño del cuadro es relativo al tamaño de la ventana del explorador o, si el cuadro está encerrado dentro de otro cuadro, es un porcentaje del tamaño del cuadro contenedor.

Cuando se utiliza ems, el tamaño del cuadro se basa en el tamaño del texto que contiene. Recientemente, los diseñadores han comenzado a usar porcentajes y ems más para las mediciones, ya que intentan crear diseños que sean flexibles en dispositivos que tienen pantallas de diferentes tamaños.

Limitación de la Altura

De la misma manera que es posible que desee limitar el ancho de un cuadro en una página, también puede limitar la altura del mismo. Esto se logra utilizando las propiedades min-height y max-height .

Si la caja no es lo suficientemente grande para contener el contenido, y el contenido se expande fuera de la caja, puede verse muy desordenado. Para controlar lo que sucede cuando no hay suficiente espacio para el contenido de un cuadro, puede usar la propiedad overflow.

Contenido desbordante

La propiedad overflow le dice al explorador qué hacer si el contenido contenido dentro de un cuadro es más grande que el propio cuadro. Puede tener uno de estos dos valores:

  • hidden

    • Esta propiedad simplemente oculta cualquier contenido adicional que no quepa en la caja.

  • scroll

    • Esta propiedad agrega una barra de desplazamiento al cuadro para que los usuarios puedan desplazarse para ver el contenido que falta.

Borde, margen y relleno

Si especifica una anchura para un cuadro, los bordes, el margen y el relleno se añaden a su anchura y altura.

Border

Cada cuadro tiene un borde (incluso si no es visible o se especifica que tiene 0 píxeles de ancho). El borde separa el borde de una caja de otra.

Margin

Los márgenes se sitúan fuera del borde de la frontera. Puede establecer la anchura de un margen para crear un espacio entre los bordes de dos cuadros adyacentes.

Padding

El relleno es el espacio entre el borde de un cuadro y cualquier contenido contenido dentro de él. Agregar relleno puede aumentar la legibilidad de su contenido.

Espacio en blanco y margen vertical

Las propiedades de relleno y margen son muy útiles para agregar espacio entre varios elementos de la página.

border-width

La propiedad border-width se utiliza para controlar el ancho de un borde. El valor de esta propiedad se puede dar en píxeles o utilizando uno de los siguientes valores:

  • thin
  • medium
  • thick

Estos valores son útiles cuando desea que los bordes de varias cajas tengan el mismo aspecto.

Puede controlar el tamaño individual de los bordes mediante cuatro propiedades independientes:

  • border-top-width
  • border-right-width
  • border-bottom-width
  • border-left-width

border-style

Puede controlar el estilo de un borde mediante la propiedad border-style. Esta propiedad puede tomar los siguientes valores:

  • solid
  • dotted
  • dashed
  • double
  • groove
  • ridge
  • inset
  • outset

border-color

La propiedad border-color puede especificar el color de un borde utilizando valores RGB, códigos hexadecimales o nombres de color CSS.

Es posible controlar individualmente los colores de los bordes en los diferentes lados de una caja usando:

  • border-top-color
  • border-right-color
  • border-bottom-color
  • border-left-color

border

La propiedad border le permite especificar el ancho, el estilo y el color de un borde en una propiedad (y los valores deben codificarse en ese orden específico).

Padding

La propiedad padding le permite especificar cuánto espacio debe haber entre el contenido de un elemento y su borde.

El valor de esta propiedad se especifica con mayor frecuencia en píxeles (aunque también es posible usar porcentajes o ems). \ Si se utiliza un porcentaje, el relleno es un porcentaje de la ventana del navegador (o del cuadro contenedor si está dentro de otro cuadro).

Tenga en cuenta: Si se especifica un ancho para una caja, se agrega relleno al ancho de la caja.

Los elementos secundarios no heredan el valor de la propiedad padding de la misma manera que el valor de color de la propiedad font-family; Por lo tanto, debe especificar el relleno para cada elemento que necesite usarlo.

Margin

La propiedad margin controla el espacio entre los cuadros. Su valor se da comúnmente en píxeles, aunque también puede usar porcentajes o ems.

Si una caja se encuentra encima de otra, los márgenes se contraen, lo que significa que se usará el mayor de los dos márgenes y no se tendrá en cuenta el más pequeño.

Tenga en cuenta: Si se especifica el ancho de un cuadro, el margen se agrega al ancho del cuadro.

Centrado de contenido

Para centrar el contenido de un cuadro, debe establecer el ancho del cuadro y luego establecer el margen izquierdo y derecho en auto.

Para centrar un cuadro en la página, debe establecer un ancho para el cuadro (de lo contrario, ocupará todo el ancho de la página).

Una vez que haya especificado el ancho del cuadro, establecer los márgenes izquierdo y derecho en automático hará que el navegador coloque un espacio igual a cada lado del cuadro. Esto centra el cuadro en la página (o dentro del elemento en el que se encuentra el cuadro).

Display

La propiedad display permite convertir un elemento en línea en un elemento de nivel de bloque o viceversa, y también se puede utilizar para ocultar un elemento de la página.

Los valores que puede tomar esta propiedad son:

  • inline

    • Esto hace que un elemento de nivel de bloque actúe como un elemento en línea.

  • block

    • Esto hace que un elemento en línea actúe como un elemento de nivel de bloque.

  • inline-block

    • Esto hace que un elemento de nivel de bloque fluya como un elemento en línea, al tiempo que conserva otras características de un elemento de nivel de bloque.

  • none

    • Esto oculta un elemento de la página. En este caso, el elemento actúa como si no estuviera en la página en absoluto.

Bordes de las Imagenes

La propiedad border-image aplica una imagen al borde de cualquier cuadro. Toma una imagen de fondo y la corta en nueve pedazos.

Posibles valores:

  • border-image-source
  • border-image-width
  • border-image-outset
  • border-image-repeat
  • border-image-slice

Sombra de los Contenedores

La propiedad box-shadow permite agregar una sombra paralela alrededor de una caja. Debe usar al menos el primero de estos dos valores, así como un color:

  • horizontal offset

    • Los valores negativos colocan la sombra a la izquierda del cuadro.

  • vertical offset

    • Los valores negativos colocan la sombra en la parte superior del cuadro.

  • blur distance

    • Si se omite, la sombra es una línea continua como un borde.

  • spread of shadow

    • Si se usa, un valor positivo hará que la sombra se expanda en todas las direcciones y un valor negativo hará que se contraiga. La palabra clave inset también se puede usar antes de estos valores para crear una sombra interna.

Bordes Redondeados

CSS3 introduce la capacidad de crear esquinas redondeadas en cualquier caja, utilizando una propiedad llamada border-radius. El valor indica el tamaño del radio en píxeles.

Puedes especificar valores individuales para cada esquina de un cuadro mediante:

  • border-top-left-radius
  • border-top-right-radius
  • border-bottom-right-radius
  • border-bottom-left-radius
