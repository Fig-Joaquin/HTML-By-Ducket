Tamaños del tipo de fuente

La propiedad font-size permite especificar un tamaño para la fuente.
Hay varias formas de especificar el tamaño de una fuente.

Los más comunes son:

Pixeles

Los píxeles se usan comúnmente porque permiten a los diseñadores web un control muy preciso sobre la cantidad de espacio que ocupa su texto.
El número de píxeles va seguido de las letras px.

Porcentajes

El tamaño predeterminado es de 16px

Em

Ems le permite cambiar el tamaño del texto en relación con el tamaño del texto en el elemento principal.

Interlineado

La propiedad line-height determina el espacio vertical entre las lineas de texto

El espacio vertical entre las líneas debe ser mayor que el espacio entre cada palabra, ya que esto ayuda a que el ojo se mueva a lo largo de la línea en lugar de hacia abajo. Un buen ajuste inicial es de alrededor de 1,4 a 1,5 em.

El texto escrito recientemente fue escrito con un line-height de 1,4em

Espaciado entre letras y palabras

letter-spacing y word-spacing dada sus definiciones, se utilizan para dar espacio entre las letras de las palabras y el espacio entre palabras.

El espacio predeterminado entre las palabras lo establece el tipo de letra (a menudo alrededor de 0,25 em), y es poco probable que necesite cambiar esta propiedad con regularidad. Si el tipo de letra está en negrita o ha aumentado el espacio entre las letras, entonces un espacio más grande entre las palabras puede aumentar la legibilidad

Alineación

La propiedad text-align permite alinear el texto a la izquierda, derecha, centrado o justificado.

Atributos para el text-align

left: Alinea el texto a la izquierda.

right: Alinea el texto a la derecha.

center: Alinea el texto al centro.

justify: Justifica el texto.

Alineación Vertical

El vertical-alignse usa más comúnmente con elementos en línea como < img>, < em>o < strong> elementos.

Sangría de Texto

La propiedad text-indent permite aplicar sangría a la primera línea de texto dentro de un elemento. La cantidad por la que desea sangrar la línea se puede especificar de varias maneras, pero generalmente se da en píxeles o ems.

Sombreado

La propiedad text-shadow

Se utiliza para crear una sombra paralela, que es una versión oscura de la palabra justo detrás de ella y ligeramente desplazada. También se puede utilizar para crear un efecto de relieve añadiendo una sombra ligeramente más clara que el texto.

Primera letra o línea

Puede especificar valores diferentes para la primera letra o la primera línea de texto dentro de un elemento mediante :first-letter y :first-line.

Link con estilos

Los navegadores tienden a mostrar los enlaces en azul con un subrayado de forma predeterminada, y cambiarán el color de los enlaces que se han visitado para ayudar a los usuarios a saber en qué páginas han estado.

:link

Esto le permite establecer estilos para los enlaces que aún no se han visitado.

:visited

Esto le permite establecer estilos para los enlaces en los que se ha hecho clic. Se utilizan comúnmente para controlar los colores de los enlaces y también si deben aparecer subrayados o no.

Estilos interactivos con el Usuario

Hay tres pseudoclases que permiten cambiar la apariencia de los elementos cuando un usuario interactúa con ellos.

:hover

Esto se aplica cuando un usuario mantiene el puntero sobre un elemento con un dispositivo señalador, como un mouse.
No funciona con dispositivos con pantalals táctiles

:active

Esto se aplica cuando un elemento está siendo activado por un usuario; por ejemplo, cuando se presiona un botón o se hace clic en un vínculo. A veces, esto se usa para hacer que un botón o enlace se sienta más como si se estuviera presionando cambiando ligeramente el estilo o la posición del elemento

:focus

Esto se aplica cuando un elemento tiene el foco. Cualquier elemento con el que pueda interactuar, como un vínculo en el que pueda hacer clic o cualquier control de formulario, puede tener el foco.

Por ejemplo, cuando el cursor está en una entrada de formulario lista para aceptar la escritura, se dice que ese elemento tiene el foco. También es posible utilizar la tecla de tabulación del teclado para desplazarse por los elementos interactivos de una página.
