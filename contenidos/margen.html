Extra markup

Atributo ID

Cada elemento de HTML puede cargar un atributo id. Se nos permite identificar entre otros elementos con un id único. Este es útil al querer darle un estilo con css

Atributo Class

Los elementos también puede llevar un atributo llamado class. Este es útil para distinguir de los elementos no tan importantes

Agrupación de texto y elementos en un bloque

El < div >

El elemento div nos permite agrupar un conjunto de elementos en un cuadro a nivel de bloque

Gracias al css se puede dar estilos a un div

El < span >

El elemento span nos permite agrupar un conjunto de elementos en un cuadro a nivel de línea

1. Contener una sección de texto en la que no haya otro elemento adecuado para diferenciarla del texto que la rodea

2. Contener una serie de elementos en línea

La razón más común por la que las personas usan < span> elementos es para poder controlar la apariencia del contenido de estos elementos usando CSS.

Iframes

Un iframe es como una ventana en tu página web que muestra otra página web. El término iframe es una abreviatura de la frase inline frame. Antes de que los iframes fueran introducidos, las páginas web no podían mostrar otras páginas web. El iframe resuelve este problema.
